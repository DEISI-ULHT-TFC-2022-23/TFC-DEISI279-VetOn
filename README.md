# Pré-Requisitos

* Download da ferramenta <a href="https://www.docker.com/get-started/">Docker Desktop</a>

# Manual de utilização  

### Container do Docker

* Pull do container
```sh
docker pull rubenulht/veton:1.0
```

* Correr o container
```sh
docker run -p 80:3000 rubenulht/veton:1.0
```

* Abrir página
```sh
localhost
```
